# poetry-speleo

## Installation

```bash
poetry self add git+https://github.com/DeviousStoat/poetry-speleo
```

## Uses

```bash
$ poetry speleo --help

Description:
  Get path to any dependency package directory in your poetry venv

Usage:
  speleo [options] [--] <package_name>

Arguments:
  package_name               The name of the package
```
